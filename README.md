# 📚 Tuition Centre Website

A modern and responsive tuition centre website designed to help students and parents learn about the institute and submit admission enquiries online. This project provides a clean landing page with all the essential information required for new admissions.

## 🌟 Features

- 🏠 Attractive Home Page
- 📖 About Tuition Centre Section
- 📚 Courses Offered
- 👨‍🏫 Experienced Faculty Information
- ⭐ 10+ Years of Teaching Experience
- 📍 Contact & Location Details
- 📝 Admission/Enquiry Form
- 📱 Fully Responsive Design
- ⚡ Fast and Lightweight Website

## 🛠️ Technologies Used

- HTML5
- CSS3
- JavaScript

## 🎯 Purpose

This website is developed to provide an online presence for a tuition centre where students and parents can:

- Learn about the tuition centre
- View available courses
- Know about the faculty
- Contact the tuition centre
- Submit admission enquiries

> **Note:** This website is intended for admission enquiries only. It does not include online payment or student management features.
